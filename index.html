Shrey Pandey

Open to ML / Agent Engineering roles

 

Bangalore, India

Hello! I’m a Machine Learning Engineer focused on production multi-agent systems, including the orchestration, evaluation, and reliability work that makes LLM agents shippable.

Most recently, I was a Machine Learning Engineer at Coinbase, building a production multi-agent system for compliance investigations using specialized LLM agents, MCP servers, and policy documents. Before that, I was a Research Fellow at Microsoft Research, where I worked on PwR, JugalBandi (featured in Satya Nadella’s keynote talks), an LLM-based Azure SDK migration framework, and dialog systems for government services. Earlier, I led ML infrastructure and ranking work at Myntra, where our SIMD-accelerated recommendations paper picked up Best Industry Paper Runner-Up at ICPE 2024.

I’m currently most interested in agent orchestration patterns, evaluation frameworks for non-deterministic systems, and the gap between agent demos and agent products. Outside work, I’m into history, geography, and the occasional culinary experiment.

If you’re hiring, collaborating, or just want to compare notes on agents, reach out on LinkedIn or at shreypandey1509@gmail.com.
