Hello! I'm a Machine Learning Engineer focused on production multi-agent systems, including the orchestration, evaluation, and reliability work that makes LLM agents shippable.

Most recently, I was a Machine Learning Engineer at [Coinbase](https://www.coinbase.com/), building a production multi-agent system for compliance investigations using specialized LLM agents, MCP servers, and policy documents. Before that, I was a [Research Fellow](https://www.microsoft.com/en-us/research/academic-program/research-fellows-program-at-microsoft-research-india/) at [Microsoft Research](https://www.microsoft.com/en-us/research/lab/microsoft-research-india/), where I worked on [PwR](https://www.microsoft.com/en-us/research/project/pwr/), [JugalBandi](https://jugalbandi.studio/) (featured in Satya Nadella's keynote talks), an LLM-based Azure SDK migration framework, and dialog systems for government services. Earlier, I led ML infrastructure and ranking work at [Myntra](https://en.wikipedia.org/wiki/Myntra), where our SIMD-accelerated recommendations paper picked up Best Industry Paper Runner-Up at [ICPE 2024](https://icpe2024.spec.org/).

I'm currently most interested in agent orchestration patterns, evaluation frameworks for non-deterministic systems, and the gap between agent demos and agent products. Outside work, I'm into history, geography, and the occasional culinary experiment.

If you're hiring, collaborating, or just want to compare notes on agents, reach out on [LinkedIn](https://www.linkedin.com/in/shrey2076/) or at [shreypandey1509@gmail.com](mailto:shreypandey1509@gmail.com).
